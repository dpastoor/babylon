## bbi run

run a (set of) models

### Synopsis


run model(s), for example: 
bbi run run001.mod
bbi run run001.mod run002.mod --cleanLvl=1  
bbi run run[001:006].mod // expand to run001.mod run002.mod ... run006.mod
bbi run . // run all models in directory
